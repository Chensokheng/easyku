### Installation & Setup
1. Clone the repositiory
```
git clone REPO-URL
```
2. Install dependencies using yarn or npm

```bash
yarn
```
or
```bash
npm install
```

3. To start development server
```
yarn start
```
or
```
npm run start
```

4. To create an optimized build:

```bash
yarn build
```
or
```
npm run build
```
